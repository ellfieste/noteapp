# Note App

A simple note-taking app for Android.

## Features
- Create, view, edit, and delete notes
- Automatic date and time stamps
- Dark/Light theme
- Support for 4 languages: English, Chinese, Russian, Greek

## Installation
1. Go to [Releases](https://github.com/ellfieste/NoteApp/releases)
2. Download the latest APK file
3. Install on Android 5.0+
